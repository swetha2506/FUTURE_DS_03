# Marketing Funnel & Conversion Analysis

## 📌 Project Overview

This project focuses on analyzing user behavior in an e-commerce platform to understand how users move through the marketing funnel:
**View → Add to Cart → Purchase**

The goal is to identify drop-off points, evaluate conversion performance, and provide actionable business insights.

## 🎯 Objectives

* Analyze user journey across funnel stages
* Calculate conversion rates between stages
* Identify major drop-offs
* Evaluate top-performing categories and brands
* Provide recommendations to improve conversions

## 🛠️ Tools Used

* Power BI
* Data Cleaning & Transformation

## 📂 Dataset

Dataset used: E-commerce user behavior dataset

🔗 Source:
https://www.kaggle.com/datasets/mkechinov/ecommerce-behavior-data-from-multi-category-store

⚠️ Note:
Due to large file size, the dataset is not uploaded in this repository.

## 📊 Key Metrics

* Total View Users
* Total Cart Users
* Total Purchase Users
* View-to-Cart Conversion Rate
* Cart-to-Purchase Conversion Rate
* Overall Conversion Rate

## 📈 Dashboard Features

* 🔻 Funnel Analysis (View → Cart → Purchase)
* 📊 Category-wise Performance
* 🏷️ Brand-wise Analysis
* 💰 Revenue Analysis
* 🎛️ Interactive Filters (Category & Brand)

## 🔍 Key Insights

* Significant drop-off observed between **View → Cart stage**
* A small percentage of users proceed to purchase after adding items to cart
* Certain product categories contribute more to purchases and revenue
* High user activity does not always translate to high conversions

## 💡 Recommendations

* Improve product page experience (images, descriptions, reviews)
* Provide discounts or offers to encourage cart additions
* Simplify checkout process to reduce cart abandonment
* Focus marketing efforts on high-performing categories

## 🚀 Conclusion

This project demonstrates how funnel analysis can help businesses understand user behavior, identify conversion bottlenecks, and make data-driven decisions to improve performance.
